IndexedHashTable.cIndexedHashTable.Add Function
Add an item to the end of the table. A unique key value is automatically created for the item. This key value will take the form "cIndexedHashTable_Key_XXXX" (where XXXX is a number from 1 to 2^63).

Public int Add (object)
  Type Name Description  
    object item The item being added to the table.  
  Return Description  
    int The position of the item in the table (0 based).
